# ----------------- TECH STACK -----------------


## Frontend
- **React** With **Vite**
- **MUI / Material UI** For easy UI components

## Database
- **PostgreSQL**

## Backend
-	**Python** with *Flask* to build the API
-	**Flask-CORS** → Allows frontend (React) to talk to the backend.
-	**Psycopg2** → Connects Flask to PostgreSQL.
